# Rectangle

### Introduction

Rectangles can be drawn as an outline, filled in, or both, and supports additional effects such as gradients, drop shadows, and dashed strokes.

<figure><img src="../../.gitbook/assets/30_05 01 26.png" alt=""><figcaption><p>Rectangle using the default white color</p></figcaption></figure>

{% hint style="info" %}
The fill, gradient, and drop shadow variable categories only appear for version 3 (or later) projects. New projects use version 3 by default.
{% endhint %}

Rectangles support all shape properties. For more information see the [Shape Properties page](shape-properties/) and subpages.
