# Line Height Multiplier

## Introduction

{% hint style="info" %}
Line Height Multiplier was added to the Gum UI tool in the January 2026 release.
{% endhint %}

Line Height Multiplier is a multiplier value which controls the spacing between each line of text. Increasing this value adds more spacing between each line of text. The default value for this is 1.
