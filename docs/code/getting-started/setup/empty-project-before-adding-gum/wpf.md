# WPF

## Introduction

Gum can be used in Skia canvases in a WPF application. To create a new WPF application, see the Create a WPF application page on Microsoft Learn: [https://learn.microsoft.com/en-us/dotnet/desktop/wpf/get-started/create-app-visual-studio](https://learn.microsoft.com/en-us/dotnet/desktop/wpf/get-started/create-app-visual-studio)

Next, you can begin adding Gum to your project. For more information see the [Adding/Initializing Gum](../adding-initializing-gum/wpf.md) page.
