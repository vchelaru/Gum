# Troubleshooting File Loading

## Introduction

This page discusses how to troubleshoot file loading.

## Troubleshooting Font Loading

Font loading can be troubleshooted by setting `GraphicalUiElement.ThrowExceptionsForMissingFiles` to `CustomSetPropertyOnRenderable.ThrowExceptionsForMissingFiles`.

For more information and example code, see the [Missing Font Exceptions](../standard-visuals/textruntime/fonts.md#missing-font-exceptions) section.
